# Mizuki-plugin-send

Mizuki Bot 的消息输出辅助插件。

本仓库用于整理机器人侧的消息输出逻辑。它通常作为其他 Mizuki 插件的配套模块使用，让不同功能可以用相对一致的格式向用户展示内容。

## 项目定位

`Mizuki-plugin-send` 更偏向输出层和适配层，不应承载复杂业务逻辑。业务插件负责生成内容，本插件负责整理展示形式。

可能包含的内容包括：

- 文本输出辅助
- Markdown 模板
- 快捷按钮内容
- 群聊与私聊场景适配
- 不同机器人实例下的输出降级

具体能力请以代码实现为准。
