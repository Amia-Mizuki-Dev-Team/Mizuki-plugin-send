# Mizuki Plugin Send

Mizuki Plugin Send 是开发组内部维护的消息输出辅助插件，用于整理机器人侧文本、Markdown、按钮和降级输出等展示逻辑。

## 项目定位

本插件偏向输出层和适配层，不应承载复杂业务逻辑。业务插件负责生成内容，本插件负责尽量统一展示形式。

典型职责包括：

- 文本输出辅助。
- Markdown 或按钮消息的公共构建。
- 不同机器人实例下的展示降级。
- 为其他插件提供更一致的消息输出体验。

## 当前状态

- 维护状态：开发组内部维护中。
- 使用范围：面向现有机器人环境和开发组插件体系。
- 文档状态：暂不补公开部署说明、完整配置项或调用方式。
- 许可证状态：暂未选择开源许可证。

## 维护注意事项

- 输出层应保持稳定，避免频繁改变其他插件依赖的消息结构。
- 富文本、按钮和纯文本降级输出应尽量保持信息一致。
- 新增模板或公共构建方法时，应检查与 help、group、welcome、economy 等插件的关系。
- 用户可见文案变更应保持克制，避免输出层掺入业务判断。
